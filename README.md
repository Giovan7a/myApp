# UniTask

Aplicativo mobile desenvolvido em React Native com Expo Router para organização de tarefas acadêmicas.

## Tecnologias utilizadas

- React Native
- Expo Go
- Expo Router
- JavaScript / TypeScript
- VS Code

## Funcionalidades

- Login de usuário
- Lista de tarefas
- Cadastro de novas tarefas
- Tela de detalhes
- Configurações
- Navegação entre telas
